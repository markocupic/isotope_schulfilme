#Erstellung eines Widgets zur Benutzung eines Attributs mit Uploadfunktion
Wir müssen zu den Produkten zur internen Verwendung Dokumente hochladen können. Dazu benötigen wir 5 Felder, die am Ende der Produkt-Bearbeiten-Seite in einem eigenen Fieldset „Dokumente“ eingefügt werden sollen:

Feldname (Dokumententyp)

 - Drehbuch Vorlage (Word, PDF)
 - Drehbuch Korrektur ( Word, PDF)
 - Drehbuch Final ( Word, PDF)
 - Übersetzungsvorlage (WORD, PDF)
 - Untertitel (SRT)

Das sollte dann so aussehen:

Feldname1 Datei-Upload-Button | Link zur Datei
Feldname2 Datei-Upload-Button | Link zur Datei
etc.

Die Dateien sollten beginnend mit der Datensatz-ID in dem Ordner „files/client/documents“ gespeichert werden.
